## 開發計畫

### 一、線上參拜 (`services/worship/index.html`)

**目標：** 互動式線上拜拜體驗，含上香動畫與擲筊功能。

**參考原型：** `/temple-map.html`（已有平面遊戲風格的三層樓廟宇地圖、神明照片彈窗、擲筊小遊戲）

**開發要點：**
- 保留統一模板格式（navbar、footer、「← 返回首頁」按鍵）
- 整合 temple-map.html 的互動地圖作為參拜入口
- 上香動畫：Canvas 煙霧粒子效果，點擊後播放上香流程
- 擲筊動畫：擲出聖筊/笑筊/陰筊，搭配音效與視覺回饋
- 點擊各神明可查看實體照片與介紹

**技術方向：**
- 純前端 HTML/CSS/JS，無框架
- Canvas API 做煙霧與擲筊動畫
- CSS 繪製平面遊戲風格廟宇場景

---

### 二、線上點燈 (`services/light/index.html`)

**目標：** 提供線上點燈（樂捐）申請表單，使用者填寫資料後匯款，廟方確認後通知。

**燈種與價格（每盞 $600）：**
- 光明燈：一般祈福，最常見
- 太歲燈：當年犯太歲者點
- 財神燈：經商求財者點

**表單欄位：**
1. 選擇燈種（光明燈 / 太歲燈 / 財神燈）
2. 點燈人姓名
3. 出生年月日（國曆）
4. 住址
5. 聯絡方式（手機 或 Email，廟方確認後通知用）
6. 匯款後五碼

**流程：**
1. 使用者選擇燈種、填寫個人資料
2. 頁面顯示廟方收款帳號資訊
3. 使用者自行匯款後，回填匯款後五碼
4. 送出表單
5. 廟方確認收款後，以簡訊或 Email 通知點燈完成

**技術方向：**
- 前端表單驗證（必填欄位、格式檢查）
- 表單送出方式待確認（目前純靜態站，可考慮 Google Forms 嵌入、Formspree、或未來接後端 API）
- 頁面設計風格與全站一致（紅金配色、宮廟莊重感）
